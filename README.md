Flutter and .NET Full-Stack Integration

This project is a demonstration of a Full-Stack Login and Registration system. It features a modern mobile frontend built with Flutter and a robust backend service developed using C# .NET.

Technical Highlights
   Backend: Developed a RESTful API with C# using ASP.NET Core. 
   Frontend: Created a responsive, themed UI in Flutter with custom styling and form validation.
   Communication: Established a secure connection via HTTP/JSON.
   Network Bridge: Configured the app to communicate via '10.0.2.2' to bridge the Android Emulator with the local development server.

 How to Run the Demo
1. Start the Backend: Open the C# folder and execute 'dotnet run'.
2. Start the Frontend: Open the Flutter folder and execute 'flutter run'.
3. Test Account: Use 'test@test.com' with password '12345' for a successful login handshake.
